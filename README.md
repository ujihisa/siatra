# Siatra: reversed art'izing tool

Art + ize `->` Artize `->` Artais `->` Siatra (reversed)

To convert your artistic code to non-artistic code.

## Install

    gem install siatra

## Usage

    $ siatra -h
    $ siatra -v
    $ siatra {filename}

or

    require 'siatra'
    p Siatra File.read '{filename}'
